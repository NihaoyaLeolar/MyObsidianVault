# Mac下安装与打开
----
## 1. 安装
Install it is using Homebrew:

```bash
brew install jmeter
```

## 2. 打开
新版的Homebrew路径在 `/opt/homebrew/`下，直接输入命令打开，就是一个用java写的工具

```
open /opt/homebrew/bin/jmeter
```
