# 编程实现
----
```java
Thread t1 = new Thread(() -> {  
    log.debug("洗水壶");  
    sleep(1);  
    log.debug("烧开水");  
    sleep(15);  
}, "老王");  
  
Thread t2 = new Thread(() -> {  
    log.debug("洗茶壶");  
    sleep(1);  
    log.debug("洗茶杯");  
    sleep(2);  
    log.debug("拿茶叶");  
    sleep(1);  
  
    //小王等待老王结束来泡茶！  
    try {  
        t1.join();     
    } catch (InterruptedException e) {  
        throw new RuntimeException(e);  
    }  
    log.debug("泡茶");  
  
}, "小王");  
  
t1.start();  
t2.start();
```
