java的双端队列有一个接口：Deque
该有两个实现类：LinkedList、ArrayDeque（一般用这两个，区别就是队列是用链表/数组实现）

# 声明与创建
---
```java
Deque<E> dq1 = new LinkedList<>();
Deque<E> dq2 = new ArrayDeque<>();
```


# 使用
---
```java
//添加
addFirst() / offerFirst()
addLast()  / offerLast()

//获取
peekFisrt();
peekLast();

//获取并取出
pollFisrt();
pollLast();
```
