>两个线程互相改变对方的结束条件，导致谁也不能结束

# 例如
---
```java
public class TestLiveLock {  

    public static int i = 10;  
    
    public static void main(String[] args) {  
        new Thread(() -> {  
            while(i > 0) {  
                sleep(1);  
                i--;  
                log.info("i:{}", i);  
            }  
        }, "t1").start();  
  
        new Thread(() -> {  
            while(i < 20) {  
                sleep(1);  
                i++;  
                log.info("i:{}", i);  
            }  
        }, "t2").start();  
    }  
}
```
# 解决方法
---
让两个线程的睡眠交错，可以采用随机的睡眠时间
